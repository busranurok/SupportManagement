# SupportManagement
Support Management


Teknik destek süreci olan firmalar için bu süreci daha ölçekli bir şekilde yönetmek ve düzenlemek adına geliştirmekte olduğum Web tabanlı bir uygulamadır.

## Kullanılan Teknolojiler
- .Net Core (API, MVC UI)
- Entity Framework
- MsSql
- N-Tier Architecture
